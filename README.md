# DevPath

DevPath is a desktop learning application designed to help beginners learn programming through a structured and guided path.

The project focuses on teaching programming step by step, where each topic builds on the previous one and gradually leads students from basic concepts to real programming skills.

## Project Goal

The goal of DevPath is to create a learning environment where programming concepts are explained through small, understandable steps, practical examples, and interactive tasks.

Instead of overwhelming students with large amounts of theory, DevPath emphasizes gradual learning, repetition, and hands-on practice.

## Learning Structure

The learning content in DevPath is organized in a hierarchical structure:

Course → Lesson → Topic → Content Blocks

Each topic contains:

• short explanations  
• code examples  
• interactive practice tasks  
• small hints and notes for beginners  

This structure allows students to progress step by step while constantly practicing what they learn.

## Current Version

DevPath is currently under active development.

Version 1 focuses on building a basic offline desktop learning environment with:

• structured programming lessons  
• topics organized into learning paths  
• progress tracking for students  
• interactive tasks with code input  
• simple code syntax highlighting  

## Technologies

This project is built using:

C#  
.NET  
WPF  
SQLite  
JSON  

SQLite is used for storing course structure and user progress.  
JSON is used for storing lesson content and interactive blocks.
